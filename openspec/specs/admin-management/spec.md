## Requirements
### Requirement: Talent Bank (Banco de Talentos)
The system MUST provide a centralized view for administrators to list and search all registered candidates in the system.

#### Scenario: Recruiter views all candidates
- **WHEN** the recruiter navigates to the "Banco de Talentos" section
- **THEN** the system displays a list of all unique candidates registered in the system
- **AND** allows filtering by name, email, or skills/tags
