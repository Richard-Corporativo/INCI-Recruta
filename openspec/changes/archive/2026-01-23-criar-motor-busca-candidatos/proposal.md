# Proposta: Motor de Busca Inteligente

## Por que
Precisamos transformar o banco de dados em um ativo ("Lake") e reduzir o custo de novas contratações reaproveitando candidatos.

## O que muda
- Nova interface de "Busca Avançada".
- Filtros por competências (tags), cargo, nível e status.
- Busca unificada entre candidatos vinculados a vagas e Banco de Talentos.

## Impacto
- **Capabilities Afetadas:** `candidate-search` (nova).
- **Código:** API de Busca (Query Builder Supabase), UI de Filtros.
