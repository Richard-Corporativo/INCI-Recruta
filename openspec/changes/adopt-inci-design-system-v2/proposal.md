# Change: Adopt INCI Design System v2.0.0

## Why
Standardization of the frontend with a modern design system compatible with shadcn/ui. This ensures visual consistency, accessibility, and a streamlined development process using tokens.

## What Changes
- **MODIFIED**: `index.html` updated with new CSS variables and Tailwind configuration.
- **MODIFIED**: `project.md` updated to include the new Design System as the source of truth.
- **IMPROVED**: All core components adjusted to use the new tokens (bg-background, text-foreground, etc.).
- **IMPROVED**: Standardized radius for buttons (base), inputs (md), and cards (lg).

## Impact
- Affected specs: `design-system`
- Affected code: `index.html`, `project.md`, all UI components.
