# Proposta: Analytics e Dashboard de Diversidade

## Por que
Faltam métricas consolidadas para visão executiva e conformidade ESG. É necessário acompanhar a saúde do funil e garantir processos inclusivos através de dados (sem rotular indivíduos).

## O que muda
- Novos widgets de KPIs (Time to Hire, Conversão, Vagas Atrasadas).
- Coleta de dados demográficos opcionais no cadastro.
- Dashboard agregado de Diversidade (anonimizado).

## Impacto
- **Capabilities Afetadas:** `analytics-system` (nova).
- **Código:** UI de Dashboard, coleta de dados no Form Candidato.
