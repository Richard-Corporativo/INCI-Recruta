# UI Strategic Playbook 2026 Skill

Uma skill especializada para criar interfaces UI/UX estratégicas baseadas no playbook 2026: "Humanizing AI for High-Conversion Ecosystems".

## 🎯 O que esta skill faz

Esta skill fornece conhecimento completo sobre design de interfaces modernas para 2026, incluindo:

### 📊 **Design Systems 2026**
- Tokens semânticos CSS com variáveis
- Sistemas de cores adaptativos (light/dark mode)
- Componentes com personalidade "adulto" e confiante
- Arquitetura de navegação otimizada (máx. 4 itens principais)

### 🧠 **Neuromarketing Aplicado**
- Princípios de psicologia da escolha para conversão
- Gatilhos mentais primais em interfaces
- Primeira impressão de 50ms
- Fluência de processamento e redução de carga cognitiva

### 💳 **Checkout Invisível**
- Fluxos de pagamento sem atrito
- Integração com Pix Automático
- BNPL integrado na jornada
- Tokenização e identidade digital

### 🎨 **Estética Imersiva 2026**
- Profundidade intencional com Glassmorphism sutil
- Natureza-inspirado "destilado"
- Retro-futurista com texturas iridescentes
- Performance vs. beleza (Limiar Doherty <400ms)

### 🔗 **Integração com Virtual Phronesis**
- Comandos específicos para design estratégico
- Análise de consistência e alinhamento
- Síntese de padrões emergentes
- Framework phronesis-driven para design

## 🚀 Quando usar

Use esta skill quando precisar:

1. **Criar sistemas de design modernos** que sigam princípios de 2026
2. **Implementar neuromarketing** em interfaces para aumentar conversão
3. **Projetar fluxos de pagamento** sem atrito
4. **Auditar interfaces existentes** com critérios de conversão
5. **Aplicar princípios de acessibilidade** modernos
6. **Integrar comandos phronesis** no processo de design

## 🔧 Componentes Incluídos

### Tokens Semânticos 2026
```css
:root {
  --text-primary: #0D1B2A;
  --action-primary: #00D448;
  --radius-full: 9999px;
  --space-6: 24px;
}
```

### Botões Pill-Shape
- CTAs com border-radius: 9999px
- Altura mínima: 44px para acessibilidade
- Feedback visual e háptico

### Cards com Tintas Contextuais
- Superfícies com cores sutis para agrupamento
- Glassmorphism para modais e tooltips
- Sombras e elevação estratégicas

### Micro-interactions
- Feedback de sucesso/erro
- Animações sutis de confiança
- Simulação de haptic feedback

## 📈 Métricas de Sucesso

A skill inclui foco em métricas importantes:

1. **Primeira Impressão**: <50ms (Limiar de confiança)
2. **Performance**: <400ms (Limiar Doherty)
3. **Hit Areas**: Mínimo 44px (Acessibilidade)
4. **Navegação**: Máximo 4 itens principais (Lei de Hick)
5. **Conversão**: Otimização contínua via A/B testing

## 🎨 Princípios de Design

### Personalidade "Adulto" 2026
1. **Restraint Confiante**: Espaço em branco generoso
2. **Copy Honesto**: Linguagem transparente e humana
3. **Micro-interações Pensadas**: Confirmação sensorial imediata

### Mandatos Obrigatórios
1. Hit area mínima: 44px
2. CTA architecture: pill-shape
3. Navegação: máximo 4 itens
4. AI transparente: tooltips explicativos
5. Performance: <400ms feedback
